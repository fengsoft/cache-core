The example folders show both the library-first path and the broader Saazip suite composition story.

- [`HTTP cache example`](./http-cache/): Minimal embedded usage showing remember, stale reads, background refresh, invalidation, and metrics.
- [`Saazip suite e2e example`](./saazip-suite-e2e/): End-to-end example showing CacheCore composing with QueueFlow, EventFlow, and WebhookCore.
